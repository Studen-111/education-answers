1. Какие типы данных существуют в стандарте ECMAScript?
   - Number, String, Boolean, Null, Undefined, Symbol, BigInt, Object.
2. Что делают операторы =, ==, === ? В чем разница?
   (=) - Присваивание, (===) и (==) - строгое и нестрогое равенство.
3. Для чего нужны различные виды циклов? Как досрочно выйти из цикла? Как перейти к следующей итерации?
   - Существует множество различных видов циклов, но все они по сути делают тоже самое - применяются для того, чтобы повторить некоторое действие нужное количество раз. Цикл состоит из условия и тела цикла. Различные по строению циклы предлагают разные способы для определения начала и окончания цикла. Для различных задач программирования существуют свои операторы цикла, с помощью которых они решаются намного проще.
   - Досрочный выход из цикла - break;
   - Переход к следующей итерации - continue.
4. Что является областью видимости переменных, объявленных при помощи var/let/const, и функций?
   Область видимости переменных var ограничивается либо функцией, либо, если переменная глобальная - скриптом.
   - var - область видимости функциональная;
   - let/const - область видимости блочная.
   Область видимости функции - функция, в котором она определена, или целая программа, если объявлена уровнем выше.
5. Когда происходит инициализация переменных и функций, а когда присваивание значения? Что будет в переменной var/let после инициализации, но до присваивания?
   Если при объявлении переменной ей не было присвоено никакого значения, она будет проинициализирована как undefined.
   - var/let x; // undefiend - без присвоения;
   - var/let x = 1; // x = 1 с присвоением.
6. Что такое метод и что такое свойство объекта?
   - Все значения в JavaScript, за исключением null и undefined, содержат набор вспомогательных функций и значений, доступных «через точку». Такие функции называют «методами», а значения – «свойствами». Метод JavaScript это свойство, содержащее определение функции.
   - Свойства — это значения, связанные с объектом JavaScript. Объект JavaScript представляет собой коллекцию неупорядоченных свойств. Свойства обычно можно изменять, добавлять и удалять, но некоторые из них доступны только для чтения.
7. Как создать новую переменную и записать туда число? Какие группы значений для чисел существуют? Какие специальные значения для чисел существуют? Как получить эти специальные значения? Как эти специальные значения ведут себя в математических операциях?
   - var/let x = 1. В современном JavaScript существует два типа чисел: обычные числа и BigInt.
   - Cпециальные значения для чисел: Infinity представляет собой математическую бесконечность. NaN означает вычислительную ошибку.
   - Если число слишком большое, оно переполнит 64-битное хранилище, JavaScript вернёт Infinity. NaN является значением, представляющим не число.
8. Какие математические операции вы знаете? Что такое Math?
   - Сложение, вычитание, умножение, деление, взятие остатка от деления, возведение в степень.
   - Объект Math является встроенным объектом, хранящим в своих свойствах и методах различные математические константы и функции.
9. Расскажите про parseInt, parseFloat? Для чего они нужны и как работают?
   - Функции parseInt, parseFloat читают число из строки. Если в процессе чтения возникает ошибка, они возвращают полученное до ошибки число. Функция parseInt возвращает целое число, а parseFloat возвращает число с плавающей точкой.
10. Как округлить число до целого? Как округлить число до 3 знаков после запятой? Как округлить до ближайшего целочисленного вверх и вниз?
    - Math.round - Округление до ближайшего целого.
    - Метод toFixed(n) округляет число до n знаков после запятой и возвращает строковое представление результата.
    - Math.ceil - Округление в большую сторону.
    - Math.flour - Округление в меньшую сторону.

    - Все остальные значения становятся true.
