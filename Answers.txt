1. Какие типы данных существуют в стандарте ECMAScript?
   - Number, String, Boolean, Null, Undefined, Symbol, BigInt, Object.
2. Что делают операторы =, ==, === ? В чем разница?
   (=) - Присваивание, (===) и (==) - строгое и нестрогое равенство.
3. Для чего нужны различные виды циклов? Как досрочно выйти из цикла? Как перейти к следующей итерации?
   - Существует множество различных видов циклов, но все они по сути делают тоже самое - применяются для того, чтобы повторить некоторое действие нужное количество раз. Цикл состоит из условия и тела цикла. Различные по строению циклы предлагают разные способы для определения начала и окончания цикла. Для различных задач программирования существуют свои операторы цикла, с помощью которых они решаются намного проще.
   - Досрочный выход из цикла - break;
   - Переход к следующей итерации - continue.
4. Что является областью видимости переменных, объявленных при помощи var/let/const, и функций?
   Область видимости переменных var ограничивается либо функцией, либо, если переменная глобальная - скриптом.
   - var - область видимости функциональная;
   - let/const - область видимости блочная.
   Область видимости функции - функция, в котором она определена, или целая программа, если объявлена уровнем выше.
5. Когда происходит инициализация переменных и функций, а когда присваивание значения? Что будет в переменной var/let после инициализации, но до присваивания?
   Если при объявлении переменной ей не было присвоено никакого значения, она будет проинициализирована как undefined.
   - var/let x; // undefiend - без присвоения;
   - var/let x = 1; // x = 1 с присвоением.
6. Что такое метод и что такое свойство объекта?
   - Все значения в JavaScript, за исключением null и undefined, содержат набор вспомогательных функций и значений, доступных «через точку». Такие функции называют «методами», а значения – «свойствами». Метод JavaScript это свойство, содержащее определение функции.
   - Свойства — это значения, связанные с объектом JavaScript. Объект JavaScript представляет собой коллекцию неупорядоченных свойств. Свойства обычно можно изменять, добавлять и удалять, но некоторые из них доступны только для чтения.
7. Как создать новую переменную и записать туда число? Какие группы значений для чисел существуют? Какие специальные значения для чисел существуют? Как получить эти специальные значения? Как эти специальные значения ведут себя в математических операциях?
   - var/let x = 1. В современном JavaScript существует два типа чисел: обычные числа и BigInt.
   - Cпециальные значения для чисел: Infinity представляет собой математическую бесконечность. NaN означает вычислительную ошибку.
   - Если число слишком большое, оно переполнит 64-битное хранилище, JavaScript вернёт Infinity. NaN является значением, представляющим не число.
8. Какие математические операции вы знаете? Что такое Math?
   - Сложение, вычитание, умножение, деление, взятие остатка от деления, возведение в степень.
   - Объект Math является встроенным объектом, хранящим в своих свойствах и методах различные математические константы и функции.
9. Расскажите про parseInt, parseFloat? Для чего они нужны и как работают?
   - Функции parseInt, parseFloat читают число из строки. Если в процессе чтения возникает ошибка, они возвращают полученное до ошибки число. Функция parseInt возвращает целое число, а parseFloat возвращает число с плавающей точкой.
10. Как округлить число до целого? Как округлить число до 3 знаков после запятой? Как округлить до ближайшего целочисленного вверх и вниз?
    - Math.round - Округление до ближайшего целого.
    - Метод toFixed(n) округляет число до n знаков после запятой и возвращает строковое представление результата.
    - Math.ceil - Округление в большую сторону.
    - Math.flour - Округление в меньшую сторону.
11. Как создать новую переменную с каким-нибудь строковым литералом?
    - var/let str = "Hello".
12. Как вывести 3й символ в этой строке?

13. Можем ли мы заменить этот символ? Если да, то как?
    - В JavaScript строки являются неизменяемыми, поэтому только с заменой новой строки, как пример через string.replace().
14. Как явно преобразовать переменную к строке? Когда происходит неявное преобразование к строке?
    - Для явного приведения значения к строке необходимо применить к нему функцию String(). Неявное преобразование будет вызвано бинарным оператором +.
15. Как посмотреть длину строки?
    - Через свойство length.
16. Как проверить есть ли в строке какое-нибудь слово (это выражение нужно будет использовать внутри if(...) )?
    - str.indexOf().
    - string.match(регулярное выражение).
17. Как взять подстроку от строки?
    - str.substring(indexA[, indexB]).
18. Как привести строку к верхнему регистру, а потом - к нижнему?
    - С помощью методов toUpperCase() и toLowerCase().
19. Как преобразовать строку в массив символов?
    - str.split().
20. Как создать новую логическую переменную? Как явно преобразовать переменную к логическому виду? Когда происходит неявное преобразование?
    - Как литерал var/let x = true.
    - Как объект  var/let y = new Boolean(false).
    - Для явного преобразования используется двойное логическое отрицание !!value или вызов Boolean(value).
    - Для неявного - происходит в логических операциях.
21. К каким логическим значениям преобразуются все другие типы?
    - Значения, вроде 0, пустой строки, null, undefined и NaN, становятся false.
    - Все остальные значения становятся true.

23. Что такое undefined? Что такое null? Как эти типы ведут себя при преобразованию к другим типам и при сравнении с другими типами?
    - undefined является специальным значением. Согласно спецификации ECMAScript, undefined в JavaScript можно получить при доступе к неинициализированным переменным, несуществующим свойствам объекта, несуществующим элементам массива. undefined является свойством глобального объекта.
    - null - Значение null представляет отсутствие какого-либо объектного значения.
    - undefined преобразуется в NaN.
    - null преобразуется 0.
    - Значения null и undefined равны (==) друг другу и не равны любому другому значению.
    - Значения null и undefined при строгом равенстве (===) различны.
24. Что такое объект? Как создать новый объект? Как добавить свойство в объект? Как поменять свойство в объекте? Как удалить свойство из объекта? Как задать свойство в названии которого есть пробел?
    - Объекты же используются для хранения коллекций различных значений и более сложных сущностей.
    - Объект может быть создан с помощью фигурных скобок {…} с необязательным списком свойств. Свойство – это пара «ключ: значение».
    var/let obj = {};
    - Добавление свойства объекта: obj.isChecked = true.
    - Обратится по ключу объекта и поменять значение.
    - Для удаления свойства мы можем использовать оператор delete: delete obj.prop.
    - Через квадратные скобки: obj["likes birds"].
25. Что такое ссылочный тип данных? В чем отличие от примитивов? Как копируются объекты? Как склонировать объект? Как сравниваются объекты?
    - Примером ссылочных типов являются объекты. Что касается массивов и функций – они являются специализированными типами объектов, поэтому также представляют собой ссылочный тип данных.
    - Переменные делятся на два типа: примитивные и ссылочные. Примитивные типы у себя внутри хранят значение, а ссылочные – ссылку на объект.
    - Когда переменная объекта копируется – копируется ссылка, сам же объект не дублируется. Для дублирования объекта используется клонирование.
    - Для клонирования можем использовать метод Object.assign.
    - Два объекта равны только в том случае, если это один и тот же объект.
26. Что такое массив? Как создать новый пустой массив? Может ли массив содержать элементы разных типов?
    - Cтруктура данных, предназначенная для хранения упорядоченных коллекций.
    - var/let array = [];
    - Да.
27. Как добавить новый элемент в конец массива или в его начало? Что работает быстрее?
    - arr.unshift() в начало массива.
    - arr.push() в конец массива.
    - Метод push() работает быстрее.
28. Как убрать элемент из конца массива или из его начала?
    - arr.pop() удаляет последний элемент массива.
    - arr.shift() удаляет первый элемент массива.
29. Как посмотреть длину массива?
    - Через свойство length: arr.length.
30. Как проверить на каком индексе в массиве находится строка ‘555’ ? А как проверить на каком индексе находится объект {a:1}? Как проверить есть ли указанные выше элементы в массиве?
    - Метод indexOf() возвращает первый индекс, по которому данный элемент может быть найден в массиве: arr.indexOf('555').
    - Метод findIndex() возвращает индекс, на котором был найден элемент.
    - Метод includes() определяет, содержит ли массив определённый элемент, возвращая в зависимости от этого true или false.
31. Как из массива слов получить строку из слов, разделенных запятыми?
    - С помощью мнтода join: arr.join(',').
32. Как удалить из массива элемент-строку “555”?
    - Использую методы indexOf() и splice().
33. Какие методы преобразования массивов вы знаете? Какой у них синтаксис и для чего они нужны?
    - map() - вызывает функцию для каждого элемента массива и возвращает массив результатов выполнения этой функции.
    - sort() сортирует массив на месте, меняя в нём порядок элементов.
    - reverse() меняет порядок элементов на обратный.
    - split(delim) разбивает строку на массив по заданному разделителю.
    - join() создаёт строку из элементов массива.
    - reduce() применяется по очереди ко всем элементам массива и «переносит» свой результат на следующий вызов.

39. Какие браузерные события вы знаете?

    click – происходит, когда кликнули на элемент левой кнопкой мыши (на устройствах с сенсорными экранами оно происходит при касании).

41. Что такое DOM-дерево и как с ним работать? Например, как в него вставить новый элемент?
    - Document Object Model, сокращённо DOM – объектная модель документа, которая представляет все содержимое страницы в виде объектов, которые можно менять.
    Объект document – основная «входная точка».
    - let div = document.createElement("div");
      div.innerHTML = "<p>Hello, World!</p>";
      document.body.append(div);
42. Что такое window, location, document?
    - Корневой объект window, который выступает в 2 ролях:
      Во-первых, это глобальный объект для JavaScript-кода: глобальный объект предоставляет переменные и функции, доступные в любом месте программы.
      Во-вторых, он также представляет собой окно браузера и располагает методами для управления им.
    - Объект location содержит информацию о расположении текущей веб-страницы: URL, информацию о сервере, номер порта, протокол.
    - Все операции с DOM начинаются с объекта document. Это главная «точка входа» в DOM. Из него мы можем получить доступ к любому узлу.
